# Basic Keylogger

A simple Python project that records keyboard keystrokes for educational and ethical cybersecurity learning purposes.

## Features
- Records keyboard keystrokes
- Logs special keys (Enter, Space, Shift, etc.)
- Saves keystrokes with timestamps
- Stores logs in a text file
- Easy-to-understand Python implementation

## Technologies Used
- Python
- pynput
- logging

## How to Run
1. Install the required library:
   pip install pynput
2. Run the Python program.
3. Type using your keyboard.
4. Press the ESC key to stop logging.
5. View the recorded keystrokes in the log file.

## Educational Purpose
This project was developed solely for learning keyboard event handling and Python logging. It should only be used on systems you own or have explicit permission to test.

## Learning Outcomes
- Keyboard event handling
- Python logging
- File handling
- Cybersecurity fundamentals
